- ログイン
- p5.jsのビジュアルコーディング
- クレジットカード情報の登録機能
- お問合せフォームの作成(AWS SESを利用してメール配信)

- SMS認証の実施

- NEWSAPIよりNEWS記事を取得
- GoogleMapを利用した地図検索の実施
